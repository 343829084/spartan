Spartan
=======

Sparan is a collection of components for High-Frequency trading. It's
goal is to provide components for exchange connectivity and
simulation. 

Components
----------

 * A feed handler for order-by-order feeds.
   
Protocols
---------
 
 * NASDAQ ITCH 5
